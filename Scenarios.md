# Scenarios  
A structured set of 25 operational scenarios covering all stakeholder groups. Each scenario is defined by a trigger, the action taken, and the expected outcome. Scenarios are traceable to their source user card via the UC reference.

---

## SC-01

| Field | Detail |
|---|---|
| **ID** | SC-01 |
| **Actor** | Board of Directors |
| **User Card** | UC-01 |
| **Domain** | Executive |
| **Trigger** | Annual ESG report is submitted for board review. |
| **Action** | Board reviews Scope 1 and 2 emission trends against prior year targets and approves updated sustainability goals for the next period. |
| **Outcome** | Updated ESG targets are formally ratified and communicated to the CEO for cascade. |
